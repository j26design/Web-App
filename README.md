# Web App

A public place to request features and report bugs for the Web App widget built for Adobe Muse.

## Bugs/Features

### Reporting bugs

1. Create a [*New Issue*](https://github.com/j26design/Web-App/issues/new?labels=bug) in the *Issues* tab.
2. In the *Labels* panel, click the gear and select the *bug* label from the dropdown.
3. Give your issue a title and add a description *(optional)* and click *Submit new issue*!

### Requesting features

1. Create a [*New Issue*](https://github.com/j26design/Web-App/issues/new?labels=enhancement) in the *Issues* tab.
2. In the *Labels* panel, click the gear and select the *enhancement* label from the dropdown.
3. Give your issue a title and add a description *(optional)* and click *Submit new issue*!
